[1] Básicos do HTML

Corpo de um HTML

‹!DOCTYPE html›
        ‹html›
              ‹head›
              ‹title›(Text Goes Here)‹/title›
              ‹/head›
                    ‹body›
                                
                    ‹/body›
        ‹/html›
    


‹!-- Atenção! Os símbolos a serem utilizados corretamente
são os de menor que e maior que ("<>"). NÃO COPIE E COLE
OS SÍMBOLOS USADOS AQUI! Eles são diferentes! --›
‹!DOCTYPE html›: Esta é uma declaração que informa ao navegador 
que o documento deve ser interpretado como HTML5, 
a versão mais recente da linguagem de marcação HTML.
    
‹html›‹/html›: Este é o elemento raiz de um documento HTML. 
Ele envolve todo o conteúdo da página da web e indica que 
o conteúdo é escrito em HTML. 
    
‹head›‹/head›: Este elemento contém metadados sobre o documento HTML, 
como título, links para folhas de estilo, scripts e outras 
informações que não são diretamente exibidas na página da web.
    
‹title›‹/title›: Este elemento define o título do documento, que é 
exibido na barra de título do navegador ou na guia do navegador.
    
‹body›‹/body›: Este elemento contém todo o conteúdo visível da página da 
web, como texto, imagens, links, vídeos, etc. Tudo o que é exibido na 
janelado navegador está contido dentro do elemento ‹body›.

Estruturas Básicas


‹h1›‹/h1›: É como um título principal de um livro. É a representação 
mais ampla e importante do conteúdo do seu livro. Por exemplo, ‹h1› 
seria o título da capa do livro, "O Mundo dos Animais".
    
‹h2›‹/h2›: É como um capítulo. Ele divide seu conteúdo em seções 
principais. Por exemplo, se sua página é sobre "Animais", um ‹h2›
pode ser "Mamíferos".
    
‹h3›‹/h3›: É como uma subseção dentro de um capítulo. Por exemplo, 
dentro do capítulo "Mamíferos", você pode ter uma subseção ‹h3› 
sobre "Cães".
    
‹h4›‹/h4›: Pode ser usado para dividir partes menores do conteúdo 
dentro das subseções. Dentro da subseção "Cães", um ‹h4› pode ser usado
para "Raças de cães".
    
‹h5›‹/h5›: Este é o nível mais detalhado. É como dividir ainda mais o 
conteúdo dentro das partes menores. No contexto dos "Cães", um ‹h5› 
pode ser usado para diferentes características de raças específicas.

‹h6›‹/h6›: Seria uma subdivisão ainda mais detalhada do conteúdo dentro 
das partes menores. Dentro das características de raças específicas de 
cães, um ‹h6› poderia ser usado para detalhar peculiaridades genéticas 
ou comportamentais muito específicas de uma raça particular.
‹p›‹/p› (paragraph): Tag usada para definir parágrafos.

‹pre›‹/pre› (pre formatted): O texto é exibido exatamente como é 
escrito no código-fonte, incluindo espaços em branco, quebras de linha 
e formatação de texto.
    
‹br›‹/br› (break row): É uma tag vazia, usada para inserir uma quebra 
de linha (ou linha em branco) em um texto.
    
‹hr›‹/hr› (horizontal row): Insere uma linha horizontal em uma página 
HTML para separar conteúdo.
    
‹!-- (Commentary Here) --›: Usada para definir um comentário sobre o 
código do site. Não afeta nada.
