[2] Como criar/adicionar Hyperlinks

‹a›‹/a›:(anchor)Tag usada para criar um hyperlink, um elemento clicável 
que direciona o usuário para outra página da web, um arquivo, uma seção 
diferente da mesma página ou até mesmo um endereço de e-mail.
    
href=" "(Hypertext REFerence):Usado para especificar o destino do link.

target=" ": Usado para especificar onde o destino do link deve ser 
aberto. Seu valor pode ser:
    
• _self:(abrir no mesmo quadro/aba).
• _blank (abrir em uma nova aba ou janela).
• _parent (abrir no quadro pai).
• _top (abrir no corpo inteiro da janela).
    
title=" ": Usado para fornecer uma tooltip quando o cursor do mouse é 
colocado sobre o link. Ele fornece informações adicionais sobre o link, 
o que pode ser útil para os usuários entenderem o destino do link antes 
de clicar nele.
