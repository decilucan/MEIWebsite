[3] Como criar/adicionar Imagens

‹img› (image): Usada para incorporar imagens em uma página HTML. Se 
usam valores para especificar detalhes sobre a imagem que será exibida:
    
• src=" " (source): Especifica o caminho para o arquivo de imagem que 
será exibido. Pode ser um URL ou um caminho no sistema de arquivos.
    
• alt=" " (alternative): Usado para fornecer um texto alternativo para 
a imagem. É exibido quando a imagem não pode ser carregada ou quando o 
usuário está usando um leitor de tela.
    
• heigh: Define a altura da imagem.
    
• width: Define a largura da imagem.
