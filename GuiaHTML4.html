[4] Como criar/adicionar Áudios

‹audio›‹/audio›: Usada para incorporar áudios em uma página HTML. Se 
usam valores para especificar detalhes sobre a imagem que será exibida:
    
• src=" " (source src=" "): Especifica o caminho para o arquivo de áudio 
que será exibido. Pode ser um URL ou um caminho no sistema de arquivos.
    
• controls (‹audio controls›): Adiciona controles de reprodução padrão 
ao elemento de áudio, como botões de reprodução, pausa e controle de 
volume.
    
• muted (‹audio muted›): Deixa o áudio sem som quando o site iniciar.
    
• autoplay (‹audio autoplay›): Inicia a reprodução automaticamente 
quando a página carregar.
    
• loop (‹audio loop›): Repete a reprodução do áudio.
    
• type (‹source src="YourSong.mp3" type="audio/extension here"›):
Define a extensão do arquivo de áudio. Útil caso o site tenha mais de
um tipo de extensão de arquivo de áudio (para backup).
    
• preload: Especifica se e como o navegador deve carregar o arquivo 
de áudio.
