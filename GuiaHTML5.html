[5] Como criar/adicionar Vídeos

‹video›‹/video›: Usada para incorporar vídeos em uma página HTML. Se 
usam valores para especificar detalhes sobre a imagem que será exibida:
    
• src=" " (source): Especifica o caminho para o arquivo de vídeo que 
será exibido. Pode ser um URL ou um caminho no sistema de arquivos.
    
• controls (‹video controls›): Adiciona controles de reprodução padrão 
ao elemento de vídeo, como botões de reprodução, pausa e controle de 
volume.
    
• muted (‹video muted›): Deixa o vídeo sem som quando o site iniciar.
    
• autoplay (‹video autoplay›): Inicia a reprodução automaticamente 
quando a página carregar.
    
• loop (‹video loop›): Repete a reprodução do vídeo.
    
• type (‹source src="YourVideo.mp4" type="video/extension here"›):
Define a extensão do arquivo de áudio. Útil caso o site tenha mais de
um tipo de extensão de arquivo de áudio (para backup).
